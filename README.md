# React-based Portfolio

This project is a **work in progress** portfolio website built using React. It is designed to showcase skills, projects, and experience while leveraging modern web development practices.

## Project Overview

This portfolio is based on the `Create React App` boilerplate and includes the following features:
- **React Router** for navigation.
- **SCSS** for styling.
- **FontAwesome** for icons.
- **GSAP** for animations.
- **Responsive Design** to ensure compatibility across devices.
